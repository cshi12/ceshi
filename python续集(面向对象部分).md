### 面向对象
面向对象是一种编程思想,就是一种写代码的套路
	编程思想有两种:
	1. 面向对象
	2. 面向过程
都是写代码的套路,最终目的就是为了将代码书写出来,只不过过程和思考方式不太一样

面向过程:
	关注的是具体步骤的实现,所有的功能都自己书写
	亲历亲为
	定义一个个函数,最终按照顺序调用函数
面向对象:
	关注的是结果,谁(对象)能帮我做这件事
	偷懒
	找一个对象,让对象去做

### 类和对象
面向对象的核心思想是 找一个对象去帮我们处理事情
在程序代码中,对象是由类创建的

类和对象是面向对象编程思想中非常重要的两个概念
类:
	是一个抽象的概念,是由多个特征和行为相同或相似事物的统称
	泛指的,(指代多个,而不是具体的一个),
	
对象:
	是具体存在的一个事物,看得见摸得着
	是特指的(指代某一个)

#### 类的组成
```
1. 类名(给这多个事物起一个名字,在代码中,满足大驼峰命名法(每个单词首字母大写))
2. 属性(事物的特征,一般文字中的名词)
3. 方法(事物的行为,即能做什么事,一般是动词)
```
#### 类的抽象(类的设计)
```
类的抽象就是找到类的类名,属性和方法


```

#### 面向代码的步骤
1. 创建类,在定义类之前设计类
2. 创建对象,使用第一步定义的类创建对象
3. 通过对象调用方法
#### 面向对象基本代码的书写
1. 定义类
```
先定义简单的类,不包含属性,在python中定义类需要使用关键字class
方法:方法的本质是在类中定义函数,只不过,第一个参数是self

class 类名:
	def 方法名(self):
	pass
```
2. 创建对象
```
创建对象是使用类名()进行创建的.即

类名()  #创建一个对象,但是这个对象在后续中不能使用;如果想要在后续代码中继续使用,需要使用一个变量,将这个对象保存起来
变量=类名()  #这个变量中保存的是对象的地址,一般可以称为这个变量为对象

一个类可以创建很多个对象,只要出现类名()就是创建对象,每个对象的地址是不同的
```
3. 调用方法
```
对象.方法名()

比如:列表.sort()
```
